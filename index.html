Programming Projects

Real-Time Online Chat

A real-time chat platform using vanilla JavaScript and WebSocket in Node.js, allowing instant communication, customizable usernames, and message history saved in a JSON file.

AI-Powered Lego Structure Recognition

A comprehensive educational application utilizing Python, the ChatGPT API, computer vision, and responsive web design, allowing elementary students to build Lego structures and engage with AI-driven recognition and feedback while learning about AI and ML.

Python-Based NLP Assistant

Python based program leveraging sentiment analysis, web scraping, and voice recognition libraries for a comprehensive natural language model voice-asssistant. Libraries used: scikit, nltk, pyttsx3, voice-recognition, and beutiful soup.

Visualizing Quantum States and Teleportation with the Bloch Sphere

A simulation using the Bloch sphere to visualize quantum states, superpositions, quantum gates, and teleportation, helping to explore concepts like quantum entanglement and the impact of operations like Hadamard and CNOT on qubit states.

(IN DEVELOPMENT) Real-Time Motion Detection and Object Tracking with OpenCV

A real-time object tracking system using OpenCV that captures video, detects motion, applies background subtraction, and tracks moving objects with bounding boxes, displaying cropped object sections in a live feed.
