AI Pictionary Game

Brought OpenAI's ChatGPT model to life for kids to interact with and test their ability to replicate objects! The user would receive a random prompt and need to build it. They then placed it infront of the video camera and the image would get processed through OpenAI's API model to guess what the object was. As a group, we got to teach kids about computer vision and AI in general!

GitHub Link: https://github.com/jesse-flores/AI-Pictionary
